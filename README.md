🌈 Portal Bettina – Acessibilidade para Todos
“Aprendi a programar aos 45 anos para criar um mundo mais acessível para minha filha autista de 3 anos. Hoje, o aprendizado é a minha missão.”

O Portal Bettina é uma plataforma gratuita de jogos, atividades e ferramentas voltadas para crianças neurodivergentes, com foco especial em acessibilidade digital, inclusão e autonomia.

Criado por um pai/mãe desenvolvedor(a) com o coração, o projeto é uma contribuição direta para melhorar a vida de crianças que enfrentam desafios no dia a dia — como minha filha, Bettina.

💡 Sobre o Projeto
🔊 Conversão de Texto para Voz com controle inteligente (ativar/desativar).

🧠 Pensado para crianças com TEA (Transtorno do Espectro Autista).

🎨 Interface simples, amigável e inclusiva.

👆 Botões acessíveis e sem duplicações, com funcionalidades claras.

💻 Desenvolvido 100% com código aberto e ❤️.

🎯 Funcionalidades
✅ Ativação e desativação da leitura de texto (TTS).

✅ Aumento de fonte para melhor legibilidade.

✅ Modo alto contraste para pessoas com baixa visão.

✅ Interface limpa, intuitiva e com controle total de acessibilidade.
