# 🎂 Jogo de Aniversário da Betina - Versão Touch 🎂

## ✨ Nova Versão - Toque Direto! ✨

Este é um jogo especial criado para Betina, uma criança autista. O jogo foi atualizado para usar **toque direto** nos presentes coloridos, tornando-o mais intuitivo e acessível.

## 🎮 Como Jogar

### 🖱️ No Computador:
- **Clique diretamente** nos presentes coloridos na tela
- Cada clique coleta o presente e anuncia sua cor

### 📱 No Celular/Tablet:
- **Toque diretamente** nos presentes coloridos na tela
- Cada toque coleta o presente e anuncia sua cor

## 🌈 Cores Disponíveis
- 🔴 **Vermelho**
- 🟢 **Verde** 
- 🔵 **Azul**
- 🟡 **Amarelo**
- 🩷 **Rosa**
- 🔵 **Azul-claro**

## 🎵 Características Especiais

### 🔊 Sistema de Áudio:
- **Sons musicais** quando coleta presentes
- **Anúncio de cores** em português
- **Melodias de celebração** nos marcos especiais
- Botão para ligar/desligar áudio

### 🎯 Sistema de Pontuação:
- Cada presente vale pontos
- A cada 10 presentes você sobe de nível
- Celebrações especiais aos 5, 25 e 50 presentes

### 📱 Totalmente Responsivo:
- Funciona em computadores, tablets e celulares
- Interface adaptável ao tamanho da tela
- Toques otimizados para dispositivos móveis

## 🚀 Como Iniciar

### Opção 1 - Arquivo Batch:
```batch
JOGO_BETINA_TOUCH.bat
```

### Opção 2 - Abrir Diretamente:
Abra o arquivo `src/index.html` em qualquer navegador

### Opção 3 - Servidor Local:
```cmd
BETINA_INICIAR.bat
```

## 🎨 Melhorias da Versão Touch

1. **Interface Simplificada**: Sem necessidade de movimentar personagem
2. **Toque Direto**: Clique/toque diretamente nos alvos coloridos
3. **Área de Toque Generosa**: Fácil para crianças pequenas
4. **Efeitos Visuais Aprimorados**: 
   - Presentes com pulsação e brilho
   - Anéis indicativos de toque
   - Partículas coloridas ao coletar
   - Efeitos de flutuação suave

5. **Feedback Imediato**: Som e voz instantâneos ao tocar
6. **Celebrações Especiais**: Confetes e animações nos marcos

## 🔧 Características Técnicas

- **HTML5 Canvas** para gráficos suaves
- **Web Audio API** para sons musicais
- **Speech Synthesis API** para anúncio de cores
- **Touch Events** otimizados para mobile
- **Responsive Design** com CSS Grid/Flexbox
