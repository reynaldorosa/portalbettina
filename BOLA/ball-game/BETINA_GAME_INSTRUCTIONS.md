# 🎂 Jogo Especial de Aniversário da Betina 🎂

## Sobre o Jogo
Este é um jogo personalizado criado especialmente para Betina em seu aniversário! O jogo foi desenvolvido com características específicas para crianças autistas, incluindo:

### ✨ Características Autism-Friendly:
- **Cores suaves e gradientes relaxantes** - Paleta de cores calmante
- **Movimentos previsíveis** - Cristais flutuam de forma suave e consistente
- **Feedback positivo constante** - Mensagens motivacionais e celebrações
- **Ambiente controlado** - Sem elementos surpresa ou assustadores
- **Música opcional** - Sons agradáveis que podem ser desligados
- **Ritmo personalizado** - O jogo se adapta ao ritmo da criança

### 🎮 Como Jogar:
1. **Objetivo**: Coletar cristais mágicos clicando/tocando neles
2. **Pontuação**: Diferentes tipos de cristais dão pontos diferentes:
   - 💎 Cristal Normal: 10 pontos
   - 💫 Cristal Bonus: 20 pontos  
   - 🎂 Cristal de Aniversário: 25 pontos
   - ⭐ Cristal Dourado Especial: 30 pontos
3. **Conquistas**: Desbloqueie conquistas especiais conforme progride
4. **Níveis**: Avance de nível coletando mais cristais

### 🎵 Recursos Especiais:
- **Personagem Betina**: Uma versão animada da aniversariante
- **Efeitos visuais suaves**: Partículas e animações calmantes
- **Mensagens motivacionais**: Frases de encorajamento personalizadas
- **Celebrações especiais**: Efeitos únicos para cristais de aniversário
- **Interface responsiva**: Funciona em computador, tablet e celular

### 🚀 Como Executar:

#### Opção 1 - Abrir diretamente:
1. Abra o arquivo `src/index.html` em qualquer navegador
2. Clique em "Começar Aventura da Betina!"
3. Divirta-se coletando cristais!

#### Opção 2 - Com servidor local (recomendado):
```bash
# Se tiver Node.js instalado:
npm install
npm start

# Ou usando Python:
cd src
python -m http.server 8000
# Depois abra: http://localhost:8000
```
