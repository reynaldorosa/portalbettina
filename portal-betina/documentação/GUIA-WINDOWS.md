# 🚀 PORTAL BETINA - GUIA DE INICIALIZAÇÃO WINDOWS

## ⚡ INÍCIO RÁPIDO (RECOMENDADO)

### Método 1 - Duplo Clique:
```
📁 Localize o arquivo: INICIAR-AQUI.bat
🖱️ Dê duplo clique
✅ Pronto! O projeto iniciará automaticamente
```

### Método 2 - Prompt de Comando:
```cmd
1. Pressione: Windows + R
2. Digite: cmd
3. Pressione: Enter
4. Cole este comando:
   cd "c:\Users\uniqs\Desktop\PORTALBETINA\portal-betina" && npm run dev
```

## 📋 PRÉ-REQUISITOS

✅ **Node.js** - Se não tiver instalado:
- Acesse: https://nodejs.org
- Baixe a versão LTS
- Instale e reinicie o Windows

## 🌐 APÓS INICIAR

O projeto abrirá automaticamente no navegador em:
```
http://localhost:5173
```

Se não abrir automaticamente, copie e cole esse link no seu navegador.

## 📱 COMO USAR NO CELULAR/TABLET

1. **Conecte o dispositivo na mesma rede WiFi**
2. **No computador, procure o IP** que aparece no terminal
3. **No celular, acesse:** `http://[IP]:5173`
4. **Ou instale como PWA:**
   - No Chrome: Menu → "Adicionar à tela inicial"
   - No Safari: Compartilhar → "Adicionar à Tela de Início"

## 🛠️ SOLUÇÃO DE PROBLEMAS

### ❌ "npm não é reconhecido"
```
Solução: Instale o Node.js e reinicie o computador
```

### ❌ Porta 5173 ocupada
```
O Vite automaticamente escolherá outra porta (5174, 5175, etc.)
```

### ❌ Erro ao instalar dependências
```cmd
# Delete a pasta node_modules e tente novamente:
rmdir /s node_modules
npm install
```

### ❌ Navegador não abre automaticamente
```
Copie e cole manualmente: http://localhost:5173
```
