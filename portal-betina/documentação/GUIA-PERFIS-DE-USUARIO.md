# 👤 Guia de Perfis de Usuário - Portal Betina

## O que são Perfis de Usuário?

Os perfis de usuário permitem que múltiplas crianças ou diferentes configurações sejam salvas no Portal Betina. Cada perfil mantém suas próprias:

- ✨ Preferências de acessibilidade
- 📊 Progresso em jogos e atividades
- 🎨 Personalizações visuais
- 📈 Histórico de desempenho

## Como Usar

### 1. Acessando os Perfis
- Clique no menu principal do Portal Betina
- Selecione "👤 Perfis de Usuário"

### 2. Criando um Novo Perfil
1. **Clique em "Adicionar Perfil"** (botão com sinal +)
2. **Preencha as informações:**
   - **Nome**: Nome da criança ou identificação
   - **Idade**: Idade da criança (1-18 anos)
   - **Ícone**: Escolha um avatar divertido
   - **Cor**: Selecione a cor favorita do perfil
   - **Nível de Acessibilidade**: 
     - Básico: Poucas adaptações
     - Médio: Adaptações padrão
     - Alto: Máxima assistência
3. **Clique em "Criar Perfil"**

### 3. Editando um Perfil Existente
- Clique no botão "✏️ Editar" no perfil desejado
- Modifique as informações necessárias
- Clique em "Salvar Alterações"

### 4. Ativando um Perfil
- **Clique diretamente no card do perfil** para ativá-lo
- O perfil ativo será marcado com uma badge "Ativo" verde
- Apenas um perfil pode estar ativo por vez

### 5. Excluindo um Perfil
- Clique no botão "🗑️ Excluir" no perfil desejado
- Confirme a exclusão na janela que aparece
- ⚠️ **Atenção**: Esta ação não pode ser desfeita!

## Funcionalidades dos Perfis

### 🎯 Perfil Ativo
- O perfil ativo é aquele sendo usado no momento
- Todas as atividades e progresso são salvos no perfil ativo
- Indicado por uma badge verde "Ativo"

### 🎨 Personalização
- **16 ícones diferentes**: De rostos a animais fofos
- **10 cores de perfil**: Para identificação visual fácil
- **Configurações de acessibilidade personalizadas**

### 📊 Progresso Individual
- Cada perfil mantém seu próprio progresso
- Estatísticas e relatórios separados
- Histórico de atividades individualizadas

## Dicas Importantes

### ✅ Boas Práticas
- **Use nomes descritivos**: "Ana 6 anos", "Pedro Autismo", etc.
- **Escolha ícones apropriados**: Ajuda na identificação visual
- **Configure a acessibilidade**: Adeque às necessidades específicas
- **Ative o perfil correto**: Verifique antes de começar atividades

### ⚠️ Limitações Atuais
- **Requer conexão com banco de dados**: Funciona apenas no modo Docker
- **Armazenamento temporário**: Em desenvolvimento, dados são mantidos em memória
- **Máximo recomendado**: 5-10 perfis para melhor performance

## Resolução de Problemas

### 💾 Perfis não aparecem?
- Verifique se está executando no modo Docker
- Confirme se há conexão com o banco de dados
- Veja a mensagem no topo da tela de perfis

### 🚫 Não consegue criar perfil?
- Verifique se preencheu o nome (obrigatório)
- Confirme se há conexão com a API
- Verifique o console do navegador (F12) para erros

### 🔄 Perfil não ativa?
- Clique diretamente no card do perfil
- Aguarde alguns segundos para atualização
- Recarregue a página se necessário

### 📱 Problemas no Mobile?
- Use toque único nos botões
- Aguarde carregamento completo
- Evite cliques duplos rápidos
